Trabalhando com listas

Listas Ordenadas (OL)

Exemplo:

  1. Acordar
  2. Tomar banho
  3. Vestir-se
  4. Tomar Café
  5. Escovar os dentes

Utilizamos a tag 'ol' para iniciar uma lista enumerada. Type = [1,A,a,I,i]

Listas Não Ordenadas (UL)

Exemplo:

  • Pão
  • Leite
  • Tomate
  • Queijo

Utilizamos a tag 'ul' para indicar uma lista não ordenada. Type = [disc, circle, square]
